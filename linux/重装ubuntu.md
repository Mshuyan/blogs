# 各种安装包
```c++
apt-get update
apt-get upgrade
apt-get install vim-gui-common  
apt-get install vim-runtime  
apt-get install git
apt-get install libssl-dev


```
# vim安装
在普通用下：
```c++
tar xvf vim.tar.gz
```
# git 
在相关的项目目录下执行以下命令。
```c++
git config credential.helper store
```

# uWS 安装
```c++
 git clone https://github.com/uNetworking/uWebSockets.git
 cd uWebsockets
 make
 make install
 ```
