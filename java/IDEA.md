### 安装 maven
* https://maven.apache.org/download.cgi
下载 apache-maven-3.5.4-bin.zip

展开,放在 C:\Program Files\maven-3.5.4 下

配置环境变量
```
MAVEN_HOME=C:\Program Files\maven-3.5.4
PATH=$MAVEN_HOME\bin;
```
