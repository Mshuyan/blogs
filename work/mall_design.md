### mall
![](https://github.com/asialiugf/blogs/blob/master/image/mall_001.png) 

* 类目可以分多级，每级都会有图中ABCD区
* A区点击三角，会覆盖C区
* A区有名称（比如，当当网叫“全部商品分类”）
* B区和C区也可以有名称，但不显示。

#### 嵌套
* 是指在上级类目点击，打开后，出现下级类目，每级类目都可以有ABCD

#### 同级组合
```
家居 /	家具 /	家装 /	厨具
男装 /	女装 /	童装 /	内衣
美妆 /	个护清洁 /	宠物
```
